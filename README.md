# 📝 To-Do List

A modern and responsive To-Do List web application built using **HTML, CSS, and JavaScript**. It helps users organize daily tasks with a simple and intuitive interface.

🌐 **Live Website:** https://itssameer666.github.io/toDoList/

## ✨ Features

* ➕ Add new tasks
* ✅ Mark tasks as completed
* ❌ Delete tasks
* 🗑️ Clear all tasks
* 💾 Save tasks using Local Storage
* 🔄 Tasks remain after page refresh
* 🚫 Prevent empty tasks
* ⌨️ Add tasks by pressing the Enter key
* 📱 Fully Responsive Design
* 🎨 Clean & Modern User Interface
* ⚡ Fast and Lightweight
* 📋 Easy Task Management

## 🛠️ Built With

* HTML5
* CSS3
* JavaScript
* Local Storage API

## 💻 Installation

1. Clone the repository

```bash
git clone https://github.com/Itssameer666/toDoList.git
```

2. Open the project folder

```bash
cd toDoList
```

3. Open `index.html` in your browser.
